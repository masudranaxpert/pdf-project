# Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Overview

This is a modern PDF tools platform built with Next.js 14, featuring:
- Merge PDF
- Split PDF  
- Compress PDF
- PDF to HTML

All tools are client-side only with mock functionality for demonstration purposes.
